This project analyzes climate trends in Delhi, focusing on:
- Air Quality Index (AQI)
- Temperature 
- Rainfall Patterns

## 📁 Folder Structure
- `data/raw/`: Original downloaded datasets
- `data/processed/`: Cleaned and merged datasets
- `notebooks/`: Jupyter notebooks for analysis
